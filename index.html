Built Fort-Knox Tough
Your Data Is Sacred

AgroNexus uses enterprise-grade security so you can farm with peace of mind.

bcrypt Password Hashing

Passwords are salted and hashed with bcrypt (cost factor 12) — never stored in plain text, ever.

JWT + Refresh Tokens

Signed RS256 JWTs with 15-min expiry. Refresh tokens are rotated, revokable, and stored as secure httpOnly cookies.

Rate Limiting & Brute-Force Guard

Login attempts are rate-limited per IP. Accounts lock after 5 failed attempts with exponential backoff.

Encrypted MongoDB Atlas

Database encrypted at rest (AES-256) and in transit (TLS 1.3). Secrets stored in Vercel environment variables, never in code.

XSS & Injection Prevention

All inputs are sanitized server-side. Helmet.js headers, CSP policies, and parameterized queries prevent common attacks.

Audit Logging

Every login, transaction, and admin action is logged with IP, timestamp and user ID. Full audit trail for compliance.
